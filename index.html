DownWords

Time: 0

Today's puzzle set by Kit Cat. Submit your own DownWords puzzle here.

  • There are 6 hidden words in the grid
  • Each word runs vertically, one letter in each of the rows (top to bottom), but the letters are not necessarily connected.
  • Each letter is only used once.
  • Five of the words are connected by a theme
  • The remaining word is the theme that connects the other words

Example:

  • Wordle
    Guess the five-letter word in six tries.
  • Spelling Bee
    Create words using letters from a honeycomb.
  • Waffle
    Rearrange letters to form words in a grid pattern.
  • Quordle
    Solve four Wordle-style puzzles simultaneously.
  • Connections
    Group words into four categories.
